## 헬름 설치하기
헬름 설치는 [공식 문서](https://helm.sh/docs/intro/install/)를 참조하여 설치합니다.


## 헬름 차트 저장소 등록하기

```bash
$ helm repo add jkim https://wja30.github.io/DevOps
$ helm repo update
```
